The best Dealer Portal to add dealer into website's any page or sidebar whatever you want. The most trusted dealer solution for WordPress.

== Description ==

Dealer-Portal allows you to create a new Dealer into your website body section, sidebar section, or footer widgets anywhere you want just to copy the [dealer] shortcode and pest it.

Major features include:

* Custom post type.
* Automatically show all Dealer.
* Images.
* Location.


== Installation ==

Upload the Dealer-Portal plugin to your website, activate it, and then copy the "[dealer]" shortcode past it where you want to show output.

1, 2, 3: You're done!
